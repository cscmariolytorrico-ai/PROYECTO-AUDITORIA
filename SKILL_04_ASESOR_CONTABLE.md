## 🔍 ANÁLISIS FINANCIERO CLAVE

### **RATIOS DE LIQUIDEZ**
```
Razón Corriente = Activos Corrientes / Pasivos Corrientes
Interpretación: > 1.5 es saludable

Razón Rápida = (AC - Inventarios) / Pasivos Corrientes
Interpretación: > 1.0 es saludable
```

### **RATIOS DE SOLVENCIA**
```
Razón de Endeudamiento = Pasivo Total / Activo Total
Interpretación: < 0.6 es saludable

Razón Deuda/Capital = Pasivo Total / Patrimonio
Interpretación: < 1.0 es saludable
```

### **RATIOS DE RENTABILIDAD**
```
ROA = Utilidad Neta / Activo Total
Interpretación: > 5% es aceptable

ROE = Utilidad Neta / Patrimonio
Interpretación: > 10% es aceptable

Margen de Utilidad Neta = Utilidad Neta / Ingresos
Interpretación: Varía por sector
```

### **RATIOS DE EFICIENCIA**
```
Rotación de Activos = Ingresos / Activo Total
Rotación de Inventarios = Costo de Ventas / Inventario Promedio
Plazo Promedio de Cobro = (Cuentas por Cobrar / Ingresos) × 365
```

## 🚨 ERRORES CONTABLES COMUNES A IDENTIFICAR

### **CRÍTICOS**
- Falta de registro de compras/ventas
- Inventario no conciliado
- Cuentas bancarias no conciliadas
- Fraude contable

### **SIGNIFICATIVOS**
- Errores de clasificación
- Depreciación incorrecta
- Provisiones no realizadas
- Transacciones no autorizadas

### **MENORES**
- Errores de cálculo
- Registro extemporáneo
- Documentación incompleta
- Formatos incorrectos
